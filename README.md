# 🚀 SocialPoster.AI

让 AI 自动登录、生成并发布你的社交内容。  
集成 GPT（生成）+ Skyvern（浏览器执行）= 一键自动发帖机器人。

---

## 🌟 核心功能
- ✍️ AI 自动生成社交文案（支持 Twitter、LinkedIn、小红书等风格）
- 🌐 Skyvern 浏览器执行器自动登录网页并发布内容
- 📅 支持定时发布、草稿保存、发帖日志
- 🧠 可扩展为多平台内容调度系统
